# TaskFlow Manager

## Overview
TaskFlow Manager is a full-stack team collaboration web application where users can create projects, assign tasks, track progress, and manage teams with role-based access control.

---

## Features

### Authentication
- User Signup/Login
- JWT Authentication
- Role-based access (Admin / Member)

### Project Management
- Create projects
- View projects
- Team collaboration

### Task Management
- Create tasks
- Assign tasks
- Update task status
- Track overdue tasks

### Dashboard
- Total projects
- Total tasks
- Completed tasks
- Overdue tasks

---
